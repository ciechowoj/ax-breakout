# Description
A simple breakout clone written in Spring 2014.

# How to run
```
pip3 install pygame
python3 main.py
```

# Issues
+ In python older than 3.7 requires workaround for issue Issue20891
+ Because of the drivers issues on Linux the sound effects have a big latency
